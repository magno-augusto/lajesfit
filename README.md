# Lajes Fit

Aplicativo social fitness para registrar dieta, treinos, publicacoes e progresso diario.

## Projeto

- Nome tecnico: `lajesfit`
- Nome visivel do app: `Lajes Fit`

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Ambiente

Use `.env.example` como referencia para configurar Supabase e Strava localmente. Nao commite chaves reais, secrets ou tokens.
